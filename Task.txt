Задание 1.
Написать приложение “Я у мамы зоолог”, в котором 4 активити.
При запуске приложения стартует главная, на ней есть три кнопки (размещены вертикально в линию),
каждая из них стартует одну из оставшихся активити.
1я активити: памятка зоолога - в верхней части экрана расположена RadioGroup с двумя RadioButton,
текстовки рядом с кнопками - “Медведи” и “Пингвины”.
(Использовать материал: https://developer.android.com/guide/topics/ui/control..)
Под RadioGroup расположена TextView, в котором хранится описание выбранного животного.
В один момент времени видно только описание того животного, которое выбрано в данный момент.
(Использовать методы класса View setVisibility). Здесь корневым лейаутом должен быть ConstraintLayout.
2я активити: экран “твое любимое животное” - вверху текст “Твое любимое животное”, под ним EditText,
в который нужно ввести название животного и второй текст, в который нужно ввести комментарий
(в каждом EditText есть подсказка hint: Животное и Комментарий соответственно).
В нижней части экрана кнопка “Отправить”, при нажатии на нее появляется сообщение
“Ваш ответ (<тут имя>, <тут начало коммента>) отправлен” (Использовать класс Toast).
Здесь корневым лейаутом должен быть RelativeLayout.
3я активити: галерея: вертикально она за другой
4-5 фотографий животных, каждая с использованием ImageView с параметров adjustViewBounds true
(фотки зашиты в ресурсы в папку drawable), для того чтобы можно было листать экран использовать ScrollView.
Здесь корневым лейаутом должен быть вертикальный LinearLayout обернутый в ScrollView.

Цель задания: отточить навыки работы с Activity и базовыми вьюхами

Также в качестве материалов можно использовать Лекцию 2 Слайды 11-21.

 + 1) Вместо самодельного лейаута с кнопкой назад использовать Toolbar, создать его в отдельном
xml-файле и добавлять в лейауты активити при помощи тега include.
 + 2) В главной активити стилизовать кнопки однотипным образом используя стили
 + 3) В активити с вводом любимого животного стилизовать оба EditText однотипным образом используя стили
 + 4) Добавить на главной активити два элемента меню: настройки и "о приложении". Клик на настройки
открывает новую активити, в которой текст "Мои интересы" и несколько CheckBox с текстом ("Коты", "Собаки", "Тигры").

 (можно фотку автора в верхнем правом углу и несколько текствьюх с описанием, контакты там, что угодно по желанию)

"Сегодня мое любимое животное - <тут_животное>", которое получается при возврате с активти с выбором
животного (так кнопка меняется с "Отправить" на "Сохранить" и по ней сразу бекаемся на главную)

При реализации меню использовать методы onCreateOptionsMenu, onOptionsItemSelected
При реализации взаимодействия по любимому животному используем startActivityForResult
